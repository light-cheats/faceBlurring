# 🎭 FaceBlur — Real-Time Face Pixelation

## 📌 Описание
FaceBlur — это приложение, которое размывает (пикселизирует) лицо в реальном времени и позволяет использовать результат как виртуальную камеру в других программах (OBS, Discord, Zoom и т.д.).

---

## 🚀 Быстрый старт

### 1. Выбор камеры

Перед запуском необходимо указать ID вашей камеры в коде.

Найдите строку с выбором камеры и измените значение:

    camera_id = 0

> ⚠️ Если камера не работает — попробуйте значения 1, 2 и т.д.

### 2. Проверка работы

После запуска программы:

- Откроется окно с вашей камерой  
- Лицо будет отображаться в пикселизированном виде  

Если это работает — можно идти дальше.

---

### 3. Подключение виртуальной камеры

1. Откройте OBS (или другую программу с поддержкой виртуальной камеры)  
2. В списке устройств выберите:

        FaceBlur

3. Включите виртуальную камеру (если требуется)

---

### 4. Использование

Теперь в любой программе (Discord, Zoom, браузер и т.д.):

- Выберите камеру **FaceBlur**  
- Готово — ваше лицо будет размыто 🎉  

---

## 💡 Примечания

- Если камера не определяется — смените `camera_id`  
- Закройте другие приложения, которые могут использовать камеру  
- В OBS убедитесь, что включена Virtual Camera
